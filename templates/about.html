About the Project

"Built an end-to-end AI-powered SaaS platform for customer conversion analytics with real-time prediction, segmentation, and intelligent insights."

The platform ingests clickstream data to intelligently predict if a browsing user will convert into a paying customer. It utilizes advanced machine learning pipelines deployed behind a robust Python Flask REST backend, integrated with a MySQL database for persistent storage.

Python Flask Scikit-Learn MySQL Vanilla CSS / Bootstrap
Tech Architecture Diagram
  • Frontend: HTML/CSS/JS + GSAP
  • Backend: Flask REST API
  • Database: MySQL (XAMPP)
  • ML Ops: Pickle + Scikit-learn Models
